# LaTeX Paper Source

This directory contains the LaTeX source files for the paper on the computationally efficient bootstrap method for EZ diffusion models.

## Compilation

The paper can be compiled in two formats:

### APA Format (Draft)

```bash
make draft
```

This compiles the paper in APA format and saves it to `pdf/draft.pdf`.

### Pretty Format (Original)

```bash
make pretty
```

This compiles the paper in the original format and saves it to `pdf/pretty.pdf`.
